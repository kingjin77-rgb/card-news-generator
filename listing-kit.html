01
토큰 발행 필수
실제 블록체인에 토큰을 배포합니다
✓
필수토큰 스마트컨트랙트 배포
Solana SPL Token 또는 EVM ERC-20 컨트랙트를 메인넷에 배포합니다.
→ COIN CREATOR로 배포하기
✓
SolanaToken Metadata 등록 (이름/심볼/로고)
Metaplex Token Metadata 프로그램으로 토큰에 이름, 심볼, 로고 URI를 등록합니다.
→ Metaboss CLI 사용 · Solana Token Creator
✓
필수유동성 잠금 (Liquidity Lock)
초기 유동성의 100%를 최소 6~12개월 잠금합니다. 신뢰도의 핵심입니다.
→ Unicrypt (EVM) · Raydium (Solana)
✓
권장컨트랙트 소유권 포기 / 멀티시그
Owner 권한을 포기(Renounce)하거나 Gnosis Safe 멀티시그로 이전하여 러그풀 우려를 해소합니다.
✓
권장스마트컨트랙트 보안 감사 (Audit)
제3자 보안 감사 리포트는 CEX 상장의 핵심 조건입니다.
→ CertiK · Hacken · Solana Audits
02
공식 문서 필수
백서, 웹사이트, 소셜 미디어 채널을 구축합니다
📄
백서 생성기
프로젝트 소개, 토크노믹스, 로드맵이 포함된 전문 백서를 자동 생성합니다.
→ 백서 만들기
🌐
랜딩페이지
코인 공식 웹사이트. CoinGecko / CMC 신청 시 반드시 필요합니다.
→ 랜딩페이지 열기
🪙
코인 크리에이터
Solana / EVM 토큰을 브라우저에서 직접 배포합니다.
→ 코인 만들기
✓
필수백서 (Whitepaper) 작성
프로젝트 비전, 토크노믹스, 로드맵, 팀 정보가 포함된 공식 문서
→ 백서 생성기로 작성하기
✓
필수공식 웹사이트 운영
도메인 구매 + HTTPS 적용 필수. CoinGecko/CMC 신청 시 필수 항목입니다.
추천 호스팅: Vercel, Netlify, GitHub Pages (무료)
✓
필수소셜 미디어 채널 개설
Twitter/X, Telegram 그룹, Discord 서버는 최소 필수입니다.
선택: Reddit, Medium 블로그, YouTube
✓
필수로고 + 브랜드 에셋
512×512px PNG (투명 배경), SVG 벡터 로고, 배너 이미지 준비
CoinGecko 요구사항: 200×200px 이상 PNG
✓
권장GitHub 레포지토리 공개
컨트랙트 소스코드를 GitHub에 공개하면 신뢰도가 크게 높아집니다.
스마트컨트랙트 검증(Verify)도 Etherscan / Solana Explorer에서 진행하세요.
03
커뮤니티 구축 권장
거래소는 커뮤니티 활성도를 매우 중요하게 봅니다
CoinGecko / CMC 상장 심사 시 Twitter 팔로워, Telegram 멤버, 거래량 등 커뮤니티 지표를 핵심으로 봅니다.
✓
권장Twitter/X — 팔로워 1,000명 이상
정기적인 업데이트 트윗, 밈 콘텐츠, 파트너 RT 교환으로 성장
✓
권장Telegram — 멤버 500명 이상
활발한 채팅 그룹 유지. 스팸봇 방지 설정 필수. 영어 + 한국어 그룹 별도 운영 추천
✓
권장에어드랍 캠페인 진행
초기 커뮤니티 확보를 위한 에어드랍. Twitter 팔로우 + Telegram 참여 조건 권장
✓
선택KOL / 인플루언서 마케팅
크립토 유튜버, Twitter KOL, CryptoTwitter 인플루언서와 협업하여 노출을 확대합니다
✓
선택CoinSniper / Dextools 트렌딩
CoinSniper · DEXTools · DEX Screener — 무료 등록 후 커뮤니티 투표
04
DEX 유동성 공급 필수
탈중앙 거래소에 먼저 상장하여 거래 가능 상태를 만듭니다
DEX 상장은 신청 없이 즉시 가능합니다. 유동성만 공급하면 누구나 거래할 수 있습니다.
Raydium
DEX · Solana
• SOL + 토큰 페어로 풀 생성
• OpenBook market ID 필요
• 최소 유동성 없음
→ 풀 생성하기
Orca
DEX · Solana
• Whirlpool 집중 유동성
• UI에서 바로 풀 생성
• 거래 수수료 공유
→ Orca 풀 생성
Uniswap V3
DEX · Ethereum
• ETH + 토큰 페어
• 집중 유동성 범위 설정
• 0.05% / 0.3% 수수료
→ Uniswap 풀 생성
PancakeSwap
DEX · BSC
• BNB + 토큰 페어
• 낮은 가스비
• 한국 커뮤니티에 인기
→ PancakeSwap 풀
05
상장 신청 필수
데이터 집계 사이트 → 중소 CEX → 대형 CEX 순으로 단계적으로 진행
⚠ 대형 CEX(Binance, Upbit)는 상장 심사가 매우 엄격합니다. 먼저 소규모 CEX와 데이터 집계 사이트부터 시작하세요.

📊 데이터 집계 사이트 (무료 · 빠름)

CoinGecko
집계 사이트
• 무료 신청 가능
• 웹사이트 + 소셜 + 거래량 필요
• 심사 기간: 1~4주
→ CoinGecko 신청
CoinMarketCap
집계 사이트
• 무료 신청 가능
• CMC Pro API 필요 없음
• 심사 기간: 2~6주
→ CMC 신청
DEX Screener
DEX 집계
• DEX 유동성 공급 시 자동 등록
• 별도 신청 불필요
• 토큰 정보 업데이트 가능
→ DEX Screener

🏦 중소형 CEX

Gate.io
CEX · 중형
• 상장 비용 있음
• 심사 기간: 2~4주
• 백서 + 팀 정보 필요
→ 상장 신청
MEXC
CEX · 중형
• 커뮤니티 투표 상장 가능
• 프로젝트 심사 병행
• 한국어 지원
→ MEXC 신청
HTX (Huobi)
CEX · 대형
• 글로벌 거래량 필요
• 팀 KYC 필수
• 상장 비용 상당
→ HTX 신청
Bybit
CEX · 대형
• 강력한 한국 사용자층
• 팀 정보 + 감사 필요
• 커뮤니티 사이즈 중요
→ Bybit 신청

🏆 대형 CEX (한국)

Upbit
CEX · 한국 1위
• 심사 매우 엄격
• 감사 리포트 필수
• 법적 검토 필요
→ Upbit 공지 확인
Bithumb
CEX · 한국 2위
• 공식 상장 신청 채널
• 재무/팀 정보 제출
• 심사 기간: 수개월
→ Bithumb 문의
Binance
CEX · 글로벌 1위
• 초엄격 심사
• 대규모 거래량 필요
• 수백만 달러 상장비
→ Binance 신청
